# Attention Tracker: Detecting Prompt Injection Attacks in LLMs

## Mục tiêu:

Dự án **Attention Tracker** Nhằm phát hiện các cuộc tấn công **Prompt Injection** vào các mô hình ngôn ngữ lớn (LLMs) bằng cách theo dõi và phân tích **Các Vector Attention**. Mục tiêu chính là cung cấp một công cụ hỗ trợ kiểm tra tính an toàn và độ tin cậy của đầu vào người dùng khi tương tác với LLMs.
